# Project Artemis: Benchmarking Engine

**Status**: COMPLETED | **Stack**: Rust, PostgreSQL, RDTSC

Artemis is a benchmarking engine built to isolate "system noise" from true performance metrics. It uses CPU time-stamp counters (RDTSC) for high-precision timing.

## Core Objective

Demonstrate the ability to isolate system noise from true performance. Financial research requires determinism; Artemis ensures that latency spikes are attributed correctly to either network jitter or code inefficiency.

## Key Features

- **RDTSC Profiling**: Nanosecond-level precision.
- **Kernel Bypass**: Uses DPDK to bypass OS network stack overhead for measurement.
- **Temporal Schemas**: PostgreSQL extension for time-series anomaly detection.
